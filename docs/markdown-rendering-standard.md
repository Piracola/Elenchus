# Markdown 渲染统一标准

## 1. 适用范围

本标准适用于 Elenchus 前端所有需要渲染 Markdown 内容的场景，包括但不限于：
- 辩手发言内容
- 裁判评价内容
- 共识收敛总结
- 系统提示信息

## 2. 渲染引擎

- **核心库**：react-markdown v9+
- **插件**：remark-gfm（支持表格、任务列表等扩展语法）
- **容器类**：`.markdown-body`

## 3. 支持的 Markdown 语法

### 3.1 基础语法
- 段落：空行分隔
- 标题：`# h1` 至 `###### h6`
- 强调：`**粗体**`、`*斜体*`
- 列表：
  - 无序列表：`- 项目`
  - 有序列表：`1. 项目`
- 链接：`[文本](url)`
- 图片：`![alt](url)`
- 代码：
  - 行内代码：`` `code` ``
  - 代码块：
    ```
    code block
    ```
- 引用：`> 引用内容`
- 水平分割线：`---`

### 3.2 扩展语法（GFM）
- 表格：
  ```markdown
  | 表头1 | 表头2 |
  | ----- | ----- |
  | 内容1 | 内容2 |
  ```
- 任务列表：
  ```markdown
  - [x] 已完成
  - [ ] 未完成
  ```

## 4. 样式规范

### 4.1 全局样式
- 字体：SF Pro Display / 系统默认无衬线字体
- 行高：1.7
- 颜色：`var(--text-primary)`
- 容器：`.markdown-body`

### 4.2 标题样式
- `h1`：1.8em，底部边框，1.5em 上边距
- `h2`：1.5em，底部边框，1.5em 上边距
- `h3`：1.25em，1.5em 上边距
- `h4`：1.1em，1.5em 上边距
- `h5`：1em，1.5em 上边距
- `h6`：0.9em，`var(--text-secondary)`，1.5em 上边距

### 4.3 表格样式
- 宽度：100%
- 边框：1px `var(--border-subtle)`
- 表头：`var(--bg-tertiary)` 背景，加粗
- 单元格：左对齐，0.6em 0.8em 内边距
- 奇偶行：偶数行 `var(--bg-secondary)` 背景
- 悬停：`var(--bg-hover)` 背景

### 4.4 代码样式
- 行内代码：`var(--bg-tertiary)` 背景，0.2em 0.4em 内边距，`var(--radius-sm)` 圆角
- 代码块：`var(--bg-tertiary)` 背景，1px `var(--border-subtle)` 边框，1em 内边距，`var(--radius-md)` 圆角，水平滚动

### 4.5 引用样式
- 左侧边框：4px `var(--accent-indigo)`
- 背景：`var(--bg-tertiary)`
- 内边距：1em 1.2em
- 圆角：`0 var(--radius-md) var(--radius-md) 0`
- 颜色：`var(--text-secondary)`
