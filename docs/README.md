# Elenchus 文档导航

## 1. 快速入口

- 项目总览与启动指南：[`README.md`](../README.md)
- 后端开发文档：[`backend/README.md`](../backend/README.md)
- 前端开发文档：[`frontend/README.md`](../frontend/README.md)
- UI 概念设计：[`docs/UI概念设计/README.md`](./UI概念设计/README.md)
- Event-Driven 架构改造：[`docs/EVENT_DRIVEN_ARCHITECTURE.md`](./EVENT_DRIVEN_ARCHITECTURE.md)

## 2. 文档分层

- `README.md`：面向使用者的主文档（安装、运行、常用 API、FAQ）。
- `backend/README.md`：面向后端开发者的架构与开发说明。
- `frontend/README.md`：面向前端开发者的结构、状态流与调试说明。
- `docs/UI概念设计/`：原型图与 UI 设计说明。
- `docs/EVENT_DRIVEN_ARCHITECTURE.md`：事件驱动架构升级记录与后续演进建议。
- `docs/archive/`：历史分析与改进过程文档（保留参考价值，不作为当前实现规范）。

## 3. 阅读建议

1. 第一次接触项目：先读主文档，再看前后端 README。
2. 要做功能开发：直接进入对应端 README 和相关代码目录。
3. 要理解演进过程：最后看 `docs/archive/` 中的历史文档。
