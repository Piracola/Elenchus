# 历史文档归档

本目录用于保存项目演进过程中的分析、诊断、修复计划与阶段性总结。

> 说明：这里的内容主要反映**当时状态**，用于回溯背景；除非文档内另有明确说明，否则**不作为当前实现的唯一事实来源**。
> 当前系统说明请优先回到 [docs/README.md](../README.md)、[architecture.md](../architecture.md)、[runtime.md](../runtime.md) 与对应开发指南。

## 使用说明

- 这些文档反映的是“当时状态”，可能与当前代码有差异。
- 当前开发、排查与实现判断应优先参考主文档和代码本身。
- 如果历史文档与 `docs/` 当前文档发生冲突，以当前文档和现有代码为准。
