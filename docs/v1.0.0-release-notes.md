# Elenchus v1.0.0

Elenchus 的首个正式稳定版本。

这是一个面向普通用户的多模型 AI 辩论应用：你可以创建辩题、配置不同模型参与辩论、实时观看发言过程，并在结束后查看评分、回放与结果记录。

## 版本亮点

- **更完整的 AI 辩论体验**
  支持多角色辩论、裁判评分、陪审团评议，以及更清晰的结果展示。

- **实时流式过程可见**
  可以直接看到辩论过程中的实时输出、状态变化与运行进度，整体体验更连贯。

- **历史回放与恢复更可靠**
  支持更稳定的会话保存、历史回放和中断恢复，方便复盘长对话与实验过程。

- **支持会话资料库**
  可以为单场辩论引入参考资料，让 AI 围绕给定材料展开讨论，而不只是依赖临时提示词。

- **更强的模型与搜索配置能力**
  支持更灵活的模型参数配置与搜索能力接入，适合不同使用场景。

- **启动与发布体验更友好**
  发布包、启动流程和运行目录已整理优化，普通用户更容易上手使用。

## 适合谁使用

- 想比较不同大模型辩论表现的用户
- 想把 AI 辩论用于观点分析、论证训练或灵感碰撞的用户
- 想进行提示词、资料注入、搜索增强等实验的开发者和研究者

## 使用方式

1. 下载对应平台的发布包
2. 解压到本地目录
3. 启动程序
4. 在设置页配置你的模型提供商 API Key
5. 创建辩题并开始辩论

## 本版本包含的主要能力

- 多模型、多角色辩论
- 实时流式输出
- 裁判评分与更丰富的评分展示
- 陪审团评议
- 会话历史保存、恢复与回放
- 会话资料库 / 参考资料注入
- 搜索配置与模型参数配置
- 更统一的发布与启动体验
