# 裁判消息块背景颜色调整

## 修改说明

将裁判消息块的综合评分卡片背景从橙色/黄色渐变改为浅灰色，使界面更加柔和舒适。

## 修改文件

1. `frontend/src/components/chat/messageRow/ScoreGrid.tsx`
2. `frontend/src/components/chat/messageRow/shared.ts`

## 具体变更

### ScoreGrid.tsx

#### 综合评分卡片背景
- **修改前**：`linear-gradient(135deg, rgba(255, 149, 0, 0.14) 0%, rgba(255, 149, 0, 0.04) 100%)`
- **修改后**：`var(--bg-secondary)` （浅灰色）

#### 综合评分卡片边框
- **修改前**：`rgba(255, 149, 0, 0.18)` （橙色）
- **修改后**：`var(--border-subtle)` （浅灰色边框）

#### "加权汇总" 标签背景
- **修改前**：`rgba(255, 149, 0, 0.12)` （橙色）
- **修改后**：`var(--bg-tertiary)` （更深的浅灰色）

#### "加权汇总" 标签文字
- **修改前**：`var(--color-judge)` （橙色）
- **修改后**：`var(--text-secondary)` （灰色）

### shared.ts

#### 裁判视觉样式 border
- **修改前**：`rgba(255, 149, 0, 0.14)` （橙色）
- **修改后**：`var(--border-subtle)` （浅灰色边框）

#### 裁判视觉样式 glowTint
- **修改前**：`rgba(255, 149, 0, 0.35)` （橙色光晕）
- **修改后**：`rgba(148, 163, 184, 0.25)` （蓝灰色光晕）

## 视觉效果

### 修改前
- 裁判消息块下方有明显的橙色/黄色渐变背景
- 边框和标签都是橙色系
- 颜色较为刺眼，长时间观看容易疲劳

### 修改后
- 使用系统的浅灰色背景变量
- 整体色调更加柔和统一
- 与界面其他部分的配色更加协调
- 保持了良好的视觉层次感

## 影响范围

- ✅ 仅影响裁判消息块的显示样式
- ✅ 不影响评分数据和功能逻辑
- ✅ 诡辩模式的特殊样式保持不变
- ✅ 响应式和动画效果正常

## 兼容性

- ✅ 使用 CSS 变量，支持主题切换
- ✅ 所有浏览器兼容
- ✅ 不影响现有的自定义主题
