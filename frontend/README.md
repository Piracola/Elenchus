# Elenchus Frontend

这个目录包含 Elenchus 的 React + Vite 前端，实现会话创建、实时聊天、运行观察与历史回放界面。

## 常用命令

```bash
cd frontend
npm install
npm run dev
npm run build
npm run lint
npm run test:run
```
